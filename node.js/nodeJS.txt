Node.js
  什么是Node.js
    Node.js 是一个基于Chrome V8 引擎的JavaScript 运行环境
    Node.js 使用了一个事件驱动，非阻塞式I/O的模型，轻量又高效
    在Node.js里运行JS和在Chrome里运行JS的不同在于：
      1. Node.js没有浏览器API,即document,window等
      2. 加了许多Node.js API
  Node.js用来做什么
    搜索引擎优化 + 浏览器首屏速度优化 = 服务端渲染
    服务端渲染 + 前后端同构 = Node.js
    构建工作流 webpack
    开发工具 Visual Studion Code
    可扩展性
      使用Node.js 做复杂本地应用,可以利用JS的灵活性提供外部扩展
    客户端应用
      在已有网站的情况下需要新开发客户端应用，用Node.js客户端技术（electron）实现，最大限度复用现有工程
